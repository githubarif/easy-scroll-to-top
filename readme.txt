== Description ==



Easy Scroll To Top is a simple Wordpress plugin that enables you to easily scroll to top of your pages by clicking a button. The button will appear only when user scroll down the pages. No need to configuration, just active the plugin and use it. 
 == Installation ==


**Through Dashboard**

1. Log in to your WordPress admin panel and go to Plugins -> Add New
2. Type **easy scroll to top** in the search box and click on search button.
3. Find easy scroll to top plugin.
4. Then click on Install button and activate the plugin.
5. Now check your scrolling pages.


**Installing Via FTP**

1. Download the plugin to your Local PC.
2. Unzip the file.
3. Upload the "easy-scroll-to-top" folder into your plugins directory.
4. Log in to your WordPress admin panel and click the Plugins menu.
5. Then activate the plugin.
6. Now check your scrolling pages.
